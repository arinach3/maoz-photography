# Usage
In the website the user can view different portfolios and albums and contact the photographer.
Clients will have a username and password for a limitted amount of time in order to sign in and view their photos, choose which photos they would like for the printed album, and download photos to his local computer.
The photographer will be able to load the albums for the clients to view, and will be able to see the selected photos for the printed album.
